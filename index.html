About this page

Welcome to my responsive photo gallery, developed as part of a Coursera course in Responsive Web Design. This page is engineered with a dynamic layout that automatically adapts to your screen size.

To test it out, view this page on a mobile device, or resize your desktop browser:

  • Mobile View: Narrow the window below 600px.
  • Desktop View: Standard browser width.
  • Widescreen View: Expand the window beyond 1200px.
